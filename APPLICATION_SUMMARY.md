# Application Summary — OPB Index Options Buying Bot

**Version:** v2.53.0
**Classification:** Institutional Trading Platform — Indian Capital Markets
**Architecture:** Clean Architecture + Port/Adapter + Event-Driven

## 1. What It Is

OPB is a fully automated, broker-independent, strategy-independent, AI-model-independent institutional-grade trading platform for Indian capital markets. It is designed to trade NSE index options (NIFTY, BANKNIFTY, FINNIFTY) with multi-asset support for equities, commodities, currency, and fixed income.

## 2. What It Does

| Function | Implementation |
|----------|---------------|
| **Signal Generation** | 14-feature ML classifier (LightGBM) + IV Rank + session classifier + regime detector |
| **Risk Management** | Multi-layer: hard halt, drawdown limits, VaR, stress tests, Kelly sizing, stale data guard |
| **Order Execution** | Deterministic state machine with exactly-once guarantees, WAL journal, idempotency certifier |
| **Portfolio Management** | Mean-variance optimization, risk-parity, CVaR optimization, correlation guard |
| **Market Data** | yfinance primary, NSE API (Akamai-blocked fallback), broker WebSocket feeds |
| **Broker Integration** | Kite Connect, Angel Broking, Paper — abstracted via BrokerPort |
| **Analytics** | Greeks engine (Black-Scholes), Monte Carlo, walk-forward, slippage model, PnL attribution |
| **Reporting** | PDF reports (ReportLab), ASCII trade replay, sensitivity analysis, health checker |
| **Governance** | Constitution validation, AI governance gate, SLO/SLA tracking, release governance |
| **Certification** | Strategy/replay/paper certifiers, unified certification gate, institutional challenge |

## 4. Key Differentiators

1. **Broker-Independent** — 3+ broker adapters via BrokerPort abstraction with failover
2. **Strategy-Independent** — Plugin-based strategy framework (spread, straddle, iron condor, pure index)
3. **AI-Model-Independent** — ML Classifier with feature store, concept drift detection, SHAP explainability
4. **Exchange-Independent** — Multi-asset market data adapters for all major Indian exchanges
5. **Vendor-Independent** — No lock-in to any single data/execution provider
